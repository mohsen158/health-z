## quickstart

```bash
git clone https://github.com/mohsen158/health-z healthZ

cd healthZ
```

```bash

yarn install

```

```bash

yarn start

```

> in a second terminal window:

```bash

yarn chain

```

> in a third terminal window:

```bash

yarn deploy

```
